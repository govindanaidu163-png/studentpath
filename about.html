About StudentPath 🚀

StudentPath is a smart career guidance platform designed to help students choose the best career path after 10th.

It provides personalized recommendations based on user interests, detailed career insights, salary information, and step-by-step roadmaps to achieve career goals.

This project is built to simplify decision-making for students and provide clarity in choosing the right future path. StudentPath is built by Govinda, a passionate developer dedicated to empowering students with the right information to shape their futures.

Tech Stack Used

HTML, CSS, JavaScript, Firebase Authentication, Firestore Database

Features

  • User Login & Signup
  • Personalized Career Recommendation
  • Explore All Career Paths
  • Search & Filter System
  • Career Details with Roadmap
